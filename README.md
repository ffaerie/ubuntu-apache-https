# Ubuntu + Apache + HTTPS 个人站点搭建实战

## 🎯 项目简介
基于Ubuntu 24.04系统，从零搭建Apache2 Web服务器，配置自定义域名`www.ZXY.com`，启用HTTPS加密访问，实现HTTP自动重定向，完成服务器安全加固，是一套完整的Linux Web服务实战项目。

## ✨ 实现功能
- ✅ 部署Apache2 Web服务，配置虚拟主机绑定自定义域名
- ✅ 生成自签名SSL证书，开启HTTPS加密访问
- ✅ 配置HTTP 301永久重定向，自动跳转到HTTPS
- ✅ UFW防火墙安全配置，仅放行80/443端口，遵循最小权限原则
- ✅ 本地域名解析（hosts），实现虚拟机/宿主机双向访问
- ✅ 编写响应式精美个人首页

## 🛠️ 技术栈
- 系统环境：VMware + Ubuntu 24.04
- Web服务：Apache2
- 安全加密：SSL/TLS、自签名证书
- 网络安全：UFW防火墙
- 前端技术：HTML、Tailwind CSS、响应式设计
- 版本控制：Git、GitHub

## 📝 核心搭建流程
1.  **环境准备**：VMware安装Ubuntu 24.04，完成系统更新
2.  **Apache部署**：安装Apache2，配置虚拟主机，绑定自定义域名
3.  **HTTPS配置**：生成自签名SSL证书，启用ssl模块，配置HTTPS虚拟主机
4.  **重定向配置**：修改HTTP配置，实现301永久跳转到HTTPS
5.  **安全加固**：配置UFW防火墙，放行80/443端口，开启防火墙
6.  **本地解析**：配置hosts文件，实现自定义域名本地访问
7.  **前端页面**：编写个人首页，部署到服务器
